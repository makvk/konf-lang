# Конфигурационный транслятор (Вариант №3)

Инструмент командной строки для преобразования текста из учебного конфигурационного языка в TOML.

## Описание

Проект разработан в рамках дисциплины "Конфигурационное управление" (3 семестр, 2025/2026 учебный год). 
Инструмент принимает на вход текст на учебном конфигурационном языке и преобразует его в формат TOML.

## Быстрый старт

### Установка
```bash
    # Клонируйте репозиторий
    git clone <repo-name>
    cd konf-lang

    # Создайте виртуальное окружение (опционально)
    python -m venv .venv
    source .venv/bin/activate  # Linux/Mac
    # или .venv\Scripts\activate  # Windows

    # Установите зависимости
    pip install -r requirements.txt
```
### Использование ###
```bash
    # Из файла
    python -m cli < input.txt > output.toml

    # Из стандартного ввода
    echo 'let x = 5 !{x + 3}' | python -m cli
    # Вывод: value = 8

    # С сортировкой
    echo '!{sort(list(3, 1, 2))}' | python -m cli
    # Вывод: value = [1, 2, 3]
```

## Синтаксис входного языка ##
### Поддерживаемые конструкции: ###
- Числа: 42, 3.14, -10
- Строки: "Hello world"
- Массивы: list(1, 2, 3, ...)
- Словари: $[
    имя : значение,
    имя : значение,
    имя : значение,
    ...
]
- Имена переменных: [a-zA-Z_][a-zA-Z0-9_]*
- Объявление констант: let имя = значение
- Константные выражения: !{выражение}
- Операции: только сложение (+)
- Функции: только sort(массив)

### Примеры ###
```
    # Простое сложение
    let x = 5
    !{x + 3}

    # Сортировка массива
    !{sort(list(3, 1, 2))}

    # Словарь с вычислениями
    let port = 8080
    $[
        server : "api",
        port : !{port},
        backup : !{port + 1000}
    ]
```
